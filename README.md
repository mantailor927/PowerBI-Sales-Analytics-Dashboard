# PowerBI-Sales-Analytics-Dashboard
Interactive Sales Analytics Dashboard built in Power BI using Excel dataset.

## 📌 Project Overview

This project is an interactive Sales Analytics Dashboard built using Power BI.

The dashboard helps analyze sales performance, product demand, customer distribution, and monthly sales trends through interactive visualizations.

---

## 🛠 Tools Used

- Power BI Desktop
- Microsoft Excel

---

## 📂 Dataset

The dataset contains four related tables:

- Orders
- Customers
- Products
- Salespersons

Relationships were created between these tables using Power BI Data Model.

---

## 📈 KPIs

The dashboard includes the following KPIs:

- Total Orders
- Total Quantity
- Average Quantity
- Maximum Quantity
- Minimum Quantity

---

## 📊 Visualizations

- Top 10 Products by Quantity
- Quantity by Category
- Sales by Region
- Monthly Orders Trend
- Interactive Month Slicer
